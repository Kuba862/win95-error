git status -> status
git add . -> dodaje wszystkie zmiany w jednym commicie
git commit -m'tresc commita' -> dodaje opis do twoich zmian
git push -> wypycha twoje zmiany do głównego repozytorium
git branch -> pokazuje wszystkie aktywne gałęzie
git checkout master -> przełącza z twojej gałęzi na główną
git checkout -b moja-nazwa-brancha-bez-spacji -> tworzy nowy branch wykonujemy to na masterze
git pull -> pobiera i aktualizuje główną gałąź. Wykonujemy na masterze.
